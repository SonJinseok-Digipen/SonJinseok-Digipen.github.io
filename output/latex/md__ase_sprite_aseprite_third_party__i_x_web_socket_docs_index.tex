IXWeb\+Socket is a C++ library for Web\+Socket client and server development. It has minimal dependencies (no boost), is very simple to use and support everything you\textquotesingle{}ll likely need for websocket dev (SSL, deflate compression, compiles on most platforms, etc...). \mbox{\hyperlink{struct_h_t_t_p}{HTTP}} client and server code is also available, but it hasn\textquotesingle{}t received as much testing.

It is been used on big mobile video game titles sending and receiving tons of messages since 2017 (i\+OS and Android). It was tested on mac\+OS, i\+OS, Linux, Android, Windows and Free\+BSD. Note that the Min\+GW compiler is not supported at this point. Two important design goals are simplicity and correctness.

\mbox{\hyperlink{class_a}{A}} bad security bug affecting users compiling with SSL enabled and Open\+SSL as the backend was just fixed in newly released version 11.\+0.\+0. Please upgrade ! (more details in the \mbox{[}\href{https://github.com/machinezone/IXWebSocket/pull/250}{\texttt{ https\+://github.\+com/machinezone/\+IXWeb\+Socket/pull/250}}\mbox{]}(PR).
