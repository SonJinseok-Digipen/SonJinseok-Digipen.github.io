Library to copy/retrieve content to/from the clipboard/pasteboard.\hypertarget{md__ase_sprite_aseprite_src_clip__r_e_a_d_m_e_autotoc_md527}{}\doxysection{Features}\label{md__ase_sprite_aseprite_src_clip__r_e_a_d_m_e_autotoc_md527}
Available features on Windows, mac\+OS, and Linux (X11)\+:


\begin{DoxyItemize}
\item Copy/paste UTF-\/8 text.
\item Copy/paste user-\/defined data.
\item Copy/paste RGB/\+RGBA images. This library use non-\/premultiplied alpha RGB values.
