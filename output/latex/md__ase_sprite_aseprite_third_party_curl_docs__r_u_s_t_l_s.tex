\href{https://docs.rs/rustls/}{\texttt{ Rustls is a TLS backend written in Rust.}}. Curl can be built to use it as an alternative to Open\+SSL or other TLS backends. We use the \href{https://github.com/rustls/rustls-ffi/}{\texttt{ rustls-\/ffi C bindings}}. This version of curl depends on version v0.\+7.\+0 of rustls-\/ffi.\hypertarget{md__ase_sprite_aseprite_third_party_curl_docs__r_u_s_t_l_s_autotoc_md1171}{}\doxysection{Building with rustls}\label{md__ase_sprite_aseprite_third_party_curl_docs__r_u_s_t_l_s_autotoc_md1171}
First, \href{https://rustup.rs/}{\texttt{ install Rust}}.

Next, check out, build, and install the appropriate version of rustls-\/ffi\+: \begin{DoxyVerb}% cargo install cbindgen
% git clone https://github.com/rustls/rustls-ffi -b v0.7.0
% cd rustls-ffi
% make
% make DESTDIR=${HOME}/rustls-ffi-built/ install
\end{DoxyVerb}
 Now configure and build curl with rustls\+: \begin{DoxyVerb}% git clone https://github.com/curl/curl
% cd curl
% ./buildconf
% ./configure --with-rustls=${HOME}/rustls-ffi-built
% make
\end{DoxyVerb}
