{\bfseries{Aseprite}} is a program to create animated sprites. Its main features are\+:


\begin{DoxyItemize}
\item Sprites are composed of \href{https://www.aseprite.org/docs/timeline/}{\texttt{ layers \& frames}} as separated concepts.
\item Support for \href{https://www.aseprite.org/docs/color-profile/}{\texttt{ color profiles}} and different \href{https://www.aseprite.org/docs/color-mode/}{\texttt{ color modes}}\+: RGBA, Indexed (palettes up to 256 colors), Grayscale.
\item \href{https://www.aseprite.org/docs/animation/}{\texttt{ Animation facilities}}, with real-\/time \href{https://www.aseprite.org/docs/preview-window/}{\texttt{ preview}} and \href{https://www.aseprite.org/docs/onion-skinning/}{\texttt{ onion skinning}}.
\item \href{https://www.aseprite.org/docs/exporting/}{\texttt{ Export/import}} animations to/from \href{https://www.aseprite.org/docs/sprite-sheet/}{\texttt{ sprite sheets}}, GIF files, or sequence of PNG files (and FLC, FLI, JPG, BMP, PCX, TGA).
\item \href{https://www.aseprite.org/docs/workspace/\#drag-and-drop-tabs}{\texttt{ Multiple editors}} support.
\item \href{https://imgur.com/x3OKkGj}{\texttt{ Layer groups}} for organizing your work, and \href{https://twitter.com/aseprite/status/806889204601016325}{\texttt{ reference layers}} for rotoscoping.
\item Pixel-\/art specific tools like \href{https://imgur.com/0fdlNau}{\texttt{ Pixel Perfect freehand mode}}, \href{https://www.aseprite.org/docs/shading/}{\texttt{ Shading ink}}, \href{https://twitter.com/aseprite/status/1196883990080344067}{\texttt{ Custom Brushes}}, \href{https://twitter.com/aseprite/status/1126548469865431041}{\texttt{ Outlines}}, \href{https://imgur.com/1yZKUcs}{\texttt{ Wide Pixels}}, etc.
\item Other special drawing tools like \href{https://twitter.com/aseprite/status/1253770784708886533}{\texttt{ Pressure sensitivity}}, \href{https://twitter.com/aseprite/status/659709226747625472}{\texttt{ Symmetry Tool}}, \href{https://imgur.com/7JZQ81o}{\texttt{ Stroke and Fill}} selection, \href{https://twitter.com/aseprite/status/1126549217856622597}{\texttt{ Gradients}}.
\item \href{https://twitter.com/pixel__toast/status/1132079817736695808}{\texttt{ Tiled mode}} useful to draw patterns and textures.
\item \href{https://twitter.com/aseprite/status/1170007034651172866}{\texttt{ Transform multiple frames/layers}} at the same time.
\item \href{https://www.aseprite.org/docs/scripting/}{\texttt{ Lua scripting capabilities}}.
\item \href{https://www.aseprite.org/docs/cli/}{\texttt{ CLI -\/ Command Line Interface}} to automatize tasks.
\item \href{https://www.aseprite.org/quickref/}{\texttt{ Quick Reference / Cheat Sheet}} keyboard shortcuts (\href{https://imgur.com/rvAUxyF}{\texttt{ customizable keys}} and \href{https://imgur.com/oNqFqVb}{\texttt{ mouse wheel}}).
\item \href{https://twitter.com/aseprite/status/1202641475256881153}{\texttt{ Reopen closed files}} and \href{https://www.aseprite.org/docs/data-recovery/}{\texttt{ recover data}} in case of crash.
\item Undo/\+Redo for every operation and support for \href{https://imgur.com/9I42fZK}{\texttt{ non-\/linear undo}}.
\item \href{https://twitter.com/aseprite/status/1124442198651678720}{\texttt{ More features \& tips}}
\end{DoxyItemize}\hypertarget{md__ase_sprite_aseprite__r_e_a_d_m_e_autotoc_md518}{}\doxysection{Issues}\label{md__ase_sprite_aseprite__r_e_a_d_m_e_autotoc_md518}
