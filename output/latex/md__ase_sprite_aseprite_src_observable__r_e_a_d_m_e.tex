Library to use the observer pattern in C++11 programs with observable/observer classes or signals/slots.\hypertarget{md__ase_sprite_aseprite_src_observable__r_e_a_d_m_e_autotoc_md543}{}\doxysection{Features}\label{md__ase_sprite_aseprite_src_observable__r_e_a_d_m_e_autotoc_md543}

\begin{DoxyItemize}
\item Generate an observable notification/signal from multiple threads
\item Add/remove observers/slots from multiple threads
\item Erase/disconnect an observer/slot from the same observable notification/signal
\item \mbox{\hyperlink{class_reconnect}{Reconnect}} an observer in the same notification
