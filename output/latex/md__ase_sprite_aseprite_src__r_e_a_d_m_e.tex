When Aseprite is compiled with {\ttfamily ENABLE\+\_\+\+DEVMODE}, you have the following extra commands/features available\+:


\begin{DoxyItemize}
\item {\ttfamily F5}\+: On Windows shows the amount of used memory.
\item {\ttfamily F1}\+: Switch new/old render engine.
\item {\ttfamily Ctrl+\+F1}\+: Switch/test Screen/\+UI Scaling values.
\item {\ttfamily Ctrl+\+Alt+\+Shift+Q}\+: crashes the application in case that you want to test the anticrash feature or your need a memory dump file.
\item {\ttfamily Ctrl+\+Alt+\+Shift+R}\+: recover the active document from the data recovery store.
\item {\ttfamily aseprite.\+ini}\+: {\ttfamily \mbox{[}perf\mbox{]} show\+\_\+render\+\_\+time=true} shows a performance clock in the Editor.
\end{DoxyItemize}
