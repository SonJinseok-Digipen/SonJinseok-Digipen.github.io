\begin{DoxyItemize}
\item Trying to encode values which are unrepresentable in JSON will never result in type conversion or other magic\+: sparse arrays, tables with mixed key types or invalid numbers (NaN, -\/inf, inf) will raise an error
\item {\ttfamily null} values contained within an array or object are converted to {\ttfamily nil} and are therefore lost upon decoding
\item {\itshape Pretty} encoding is not supported, {\ttfamily json.\+encode()} only encodes to a compact format
