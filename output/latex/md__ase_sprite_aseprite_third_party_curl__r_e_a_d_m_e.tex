Curl is a command-\/line tool for transferring data specified with URL syntax. Find out how to use curl by reading \href{https://curl.se/docs/manpage.html}{\texttt{ the curl.\+1 man page}} or \href{https://curl.se/docs/manual.html}{\texttt{ the MANUAL document}}. Find out how to install Curl by reading \href{https://curl.se/docs/install.html}{\texttt{ the INSTALL document}}.

libcurl is the library curl is using to do its job. It is readily available to be used by your software. Read \href{https://curl.se/libcurl/c/libcurl.html}{\texttt{ the libcurl.\+3 man page}} to learn how!

You can find answers to the most frequent questions we get in \href{https://curl.se/docs/faq.html}{\texttt{ the FAQ document}}.
