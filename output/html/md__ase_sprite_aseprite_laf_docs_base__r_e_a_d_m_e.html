laf-base

Cross-platform core functionality to do basic tasks. Ideally this functionaly should be included on C++, so some functions could be replaced with a std:: equivalent in the future.

  • Data utilities (encode/decode_base64)
  • File system & filename/path utilities (fs.h)
  • File utilities (serialization, sha1, launcher)
  • Logging functions (LOG())
  • Manage DLLs (load/unload_dll())
  • Multi-threading utilities (thread, thread_pool)
  • Smart pointers (RefCount/Ref)
  • String/UTF8 utilities (string, split_string, trim_string)
  • Timing (tick_t/current_tick(), Chrono)
  • Type conversion (convert_to)
  • Unicode filenames (open_file_raw(), FSTREAM_PATH())
  • Version comparison (Version)
